DIBP Financial Delegations e-learning website

Financial Delegations was a project built for the Department of Immigration and Border Protection. As a government site, not only did it have to work on a wide range of browsers but it also need to comply to WCAG up to AA standards.

This was one of the first projects I worked on which did not require the use of a WYSIWYG for development (nor did any of the WYSIWYGs meet WCAG). Instead it was developed entirely by code. I started with a template which had already been created by the development team, but it also needed features which the template did not already have. The major updates I developed for the template and this project were:

  • updating all HTML and features to comply with WCAG AA standards
  • developing the site to work with IE browsers all the way down to IE6
  • developing “streaming” aka. multiple paths to completion
  • developing assessments into the template
  • upgrading the navigation code to allow for the new streaming features

The assessments code was the most interesting to write, and you can see my setup >> README here. The README explains how to setup the assessments within the templates existing COURSEPROPS object. The COURSEPROPS object was basically used like a settings page, where certain global properties were set. You can see a snippet of my >> assessments code here. It was developed into the existing template code.

Built for IE6+, Chrome, Firerox, Safari, Tablet browsers.
