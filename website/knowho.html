Knowho prototype

KnoWho is a prototype website I built for a mate’s university project which was part of a world-wide competition. The project was a new web application, and my website was going to be the advertisement for it. The project ended up in the top 5 in Australia, but sadly didn’t end up winning, hence the project was ended and the website didn’t reach completion.

As the time was limited, I decided to go with using a free responsive website template and customizing it to our needs.
