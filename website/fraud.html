CBA Fraud e-learning website

Fraud is a e-learning site built for Commonwealth Bank of Australia. It is for training employees about fraud within the workplace. This particular project was significant because it required a large upgrade to the current Lectora template being used. The project needed to include multiple languages within the one SCORM package, which then meant the site needed “streaming”, multiple assessments, a rewrite of the navigation code, plus the entire thing had to be as close to WCAG AA standards as possible.

Most of the javascript was based on the same features built into the HTML template from the >> DIBP Financial Delegations project, and adjusted to fit into the Lectora template. The features were harder to implement however, with multiple changes and work-arounds (aka. hacks) needed, because of Lectora’s own badly generated code.

In the screenshots, you can see the same page translated into 3 different languages.
Built for IE7+, Chrome, Firefox, Safari.
