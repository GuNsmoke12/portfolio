CBA Data Quality e-learning website

An e-learning website made for the Commomwealth Bank of Australia. Built using the WYSIWYG “Lectora”, this project still needed some customisation which went beyond the development limits of Lectora. This meant writing code specific for this one project, exploiting Lectora’s “custom HTML box” feature.
>> You can see my explanation of Lectora and my use with it here.

First was the implementation of Jplayer. Lectora’s default video feature only uses an older flash player which did not meet our client’s needs, so Jplayer was implemented to allow HTML5 video player capabilities with a Flash video player for fall-back. The other feature was the scroll-to page with a parallax effect.

It was built for IE7+, Chrome, Firefox, Safari, Tablet browsers.
